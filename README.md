# andtest-core
AndTest core classes. Used in AndTest android app and other apps (wip).

In project build.gradle:
```
allprojects {
  repositories { 
    jcenter()
    maven { url "https://jitpack.io" }
  }
}
```
In module build.gradle:
```
dependencies {
  compile 'com.github.yeriomin:andtest-core:0.1'
}
```
